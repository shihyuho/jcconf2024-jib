# Jib 打包 Java 應用的那些事：心得分享與擴充攻略

讓我們來聊聊 [Jib](https://github.com/GoogleContainerTools/jib)，這是個專門為 Java 開發者設計的容器打包工具。我會簡單的介紹它的優點和運作方式，並展示如何用 Jib 打包 Java 應用程式，接著再分享如何擴充 Jib 的功能，以滿足我們團隊的使用場景。希望這次分享能幫助你更好的使用 Jib！
